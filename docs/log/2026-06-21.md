# 2026-06-21

## Architectural Ideas & Editions Update

- Updated `docs/Ideas/architectural_ideas.md` with **Toolchain as Middleware (Modular Crates)**: proposed splitting the `mimz` monolith into separate Cargo workspace crates (`mimz-parser`, `mimz-checker`, `mimz-sim`, `mimz-emit`). This allows the community to swap individual components without bypassing the core `mimz-checker`, thereby preserving the "Safe-by-Default" guarantees.
- Updated `spec/06-editions.md`: specified that `lang/case_suffixes.toml`, `lang/keywords.toml`, and `lang/messages.toml` will use a history tracking file (e.g., `history.toml`) to store older or modified keywords. This ensures perfect backward compatibility for source files pinned to older language editions.
- **Refactor:** Moved the core language data files (`keywords.toml`, `messages.toml`, `case_suffixes.toml`) from the repository root into a new `lang/` folder to clean up the architecture. Updated all `include_str!` macros in `src/`, test paths in `tests/`, and documentation references accordingly.
