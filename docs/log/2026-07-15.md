# 2026-07-15

## Enum Variant Construction (`EnumConstruct`)

Shipped full end-to-end support for constructing enum variants with payloads using the `Enum.Variant(args)` syntax.

- **Parser**: Added parsing for `EnumConstruct` syntax.
- **Checker**: Validates name, variant, and arity (emitting E0103/E0806 on failure). Infers `Ty::Enum`, correctly width-checks arguments, and resolves cross-file field widths for constant-foldable arguments.
- **Emitter**: Lowers `EnumConstruct` to a clean tag+payload Verilog concatenation.
- **Simulator**: Evaluates `EnumConstruct` by lowering it to a `Concat` node at elaboration time.
- **Docs & Testing**: Synced across the spec, code guide, and changelog. Added a 5-flavor example with golden outputs and differential validation against Icarus Verilog.

## Extern Modules (Verilog FFI)

Implemented the `extern module` construct, unlocking true black-box IP integration and Verilog FFI.

- **Keywords**: Promoted `extern` to an active keyword (provisional Tamil: `anniya`/`அன்னிய`).
- **Parser**: Parses `extern module` declarations.
- **Checker**: Validates declarations, structural instantiations, and width-checks all port connections against the `extern` signature.
- **Emitter**: Skips emitting the module body, generating only the instantiation to link against external Verilog files.
- **Simulator / CLI**: Added strict/warn configuration plumbing for `extern` module simulation support, ensuring un-simulatable black boxes can be explicitly bypassed or warned during `mimz sim` execution.
