## Bundle Support Implementation (Phase 2)

- **AST & Parser**: Added Bundle declarations, parametric bundle types, bundle literals, and let destructuring support to the AST and parser.
- **Checker**: Implemented bundle registration and field type validation (E0906, E0909). Added semantic checks for bundle literals, destructuring, and nominal type matching (E0901-E0903, E0907). Applied T6 review fixes.
- **Emitter**: Added support for bundle types. Blocked bundle destructure emission and clarified tracking for repeat-body wires.
- **Simulator**: Flattened bundle-typed signals into prefixed scalars. Implemented O(1) bundle type lookup and added guards for missing fields in bundle literals.
- **Pretty-Printer**: Implemented formatting for bundle declarations, destructuring, literals, and the Type::Bundle variant.
- **Documentation**: Updated the language specification (section 1.12) with the bundle grammar delta, documented the new E-codes (E0901-E0903, E0906-E0907, E0909), and provided a four-flavor example.
