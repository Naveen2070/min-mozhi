# 2026-07-08

## Added `sim` emulation blocks to examples

We wired up the new `led` hardware emulation peripheral (added in yesterday's PR) to several existing example and showcase files. This allows users to visualize their designs running with real-time paced LEDs when using `mimz test --emulate`.

- `examples/english/blinker.mimz`: Bound the `led` output to a red LED (`speed mhz(50)`).
- `examples/english/traffic_light.mimz`: Bound `red`, `yellow`, and `green` to their respective LED colors (`speed hz(10)`).
- `examples/english/vilakku.mimz` and `examples/tamil/vilakku.mimz`: Bound `ஒளி` to a yellow LED (`speed hz(2)`), fully utilizing Tamil keywords in the Tamil flavor.
- `demo/cpu.mimz`: Bound the `leds` bus to a red LED (`speed hz(2)`).

These changes demonstrate the new hardware emulation capabilities on the existing codebase without modifying the core functionality.
