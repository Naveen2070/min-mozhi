## foreach sugar — implementation

- Parser: `foreach` keyword + `in` separator, range (`0..4`) and element-form sources, module-item and `on`-block/`fn`-body placement.

- Emit/sim/pretty/translit/lint: each lower on the spot — no new codegen shape.
- `foreach_lower.rs` (1163 lines): shared lowering logic consumed by all backends.
- Five-flavor examples (`foreach_fill`, `foreach_sum`) + Verilog goldens + Icarus differential + E0417 error fixture.
- 764 tests, clippy clean.
